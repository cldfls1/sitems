В админ панели показывает "No items yet" - база данных пустая

РЕШЕНИЕ ДОБАВЛЕНО!

═══════════════════════════════════════════════════════════

✅ ЧТО ДОБАВЛЕНО:

1. Backend endpoint: /api/init-data
   - Загружает демо данные (проекты, навыки, about, content)
   - Проверяет, есть ли уже данные (не дублирует)

2. Кнопка в админке: "Load Demo Data"
   - Появляется когда нет данных
   - Один клик - все загрузится!

3. API функция: initializeData()
   - Вызывает endpoint
   - Обрабатывает ошибки

═══════════════════════════════════════════════════════════

🚀 КАК ИСПОЛЬЗОВАТЬ:

ВАРИАНТ 1: Через админку (ПРОЩЕ ВСЕГО)
1. Открой админ панель
2. Увидишь "No items yet"
3. Нажми кнопку "Load Demo Data" 
4. Подтверди
5. Готово! Данные загружены 🎉

ВАРИАНТ 2: Через API напрямую
curl -X POST https://твой-домен.vercel.app/api/init-data
